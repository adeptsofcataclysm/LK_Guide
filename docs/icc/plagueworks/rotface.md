## Гниломорд

<center>

![Gniloe_ebalo2](/img/icc/Rotface/Rotface.png)
</center>

Гниломорд — один из босов-поганищ Чумодельни Цитадели Ледяной Короны.
Его убийство требуется для открытия прохода к Профессору Мерзоциду.
